# 縮圖 Brief — Stephanie Lazarus / Sherri Rasmussen 命案

> WF4 production-package 產出。3 個方向供挑選。視覺依 visual-style（暗色、高對比、非真實人臉）。
> 本案核心鉤子：**殺人者是警察**、**咬痕 DNA 破案**、**冷案沉睡 23 年**。三方向各押一個鉤子。

## 方向 1 — 「警徽下的兇手」（身分反差）
- 主視覺：暗色背景中一枚 LAPD 風格金屬警徽特寫，警徽金屬面浮現一道裂縫／血漬暗紋；前景一抹無臉女性剪影（背光、看不見五官）疊在警徽陰影上。強焦點＝警徽。
- 大標（4–8字）：兇手是警察
- 配色：墨黑底 × 冷藍灰金屬 × 警示橘點綴（裂縫描邊）；標題用冷白＋橘。
- 生成提示詞（英文）：
```
cinematic documentary still, dark moody lighting, cold desaturated palette, film grain,
extreme close-up of a tarnished generic police detective badge (no readable text, no real insignia),
a faint crack and dark stain running across the metal, an anonymous backlit female silhouette
faintly overlaid in shadow, face not visible, high contrast, single strong focal point,
investigative crime tone, 16:9 --ar 16:9
```

## 方向 2 — 「咬痕裡的真相」（鑑識物證）
- 主視覺：暗色法醫工作檯，一張被害人手臂示意石膏／矽膠模型上的咬痕特寫（去識別、非真實人臉、非血腥），旁邊一支試管／棉棒採樣與一張模糊冷案卷宗。強焦點＝咬痕＋一管發光的 DNA 採樣。
- 大標：咬痕 DNA
- 配色：墨黑／深藍灰底 × 檔案暖黃（卷宗）× 冷白標題；採樣管用低調青綠冷光。
- 生成提示詞（英文）：
```
cinematic documentary still, dark moody lighting, cold desaturated palette, film grain,
forensic evidence table top-down, a pale silicone cast of a human forearm showing a bite-mark
impression (anonymous, no face, not gory), a DNA swab and a glowing sample tube beside it,
an old blurred case file folder with redacted papers, shallow depth of field, high contrast,
single strong focal point, investigative forensic tone, 16:9 --ar 16:9
```

## 方向 3 — 「沉睡 23 年的冷案」（時間／冷案）
- 主視覺：昏暗證物室／檔案室，積灰的紙箱裡一份貼著舊標籤的命案卷宗，標籤上大字年份「1986」清晰；背景是 1980 年代 Van Nuys 夜間公寓外景剪影（時代正確、無現代地標），門口一條警示膠帶。強焦點＝積灰卷宗＋年份。
- 大標：沉睡23年
- 配色：墨黑底 × 冷藍灰 × 檔案暖黃舊紙；年份與標題用警示橘＋冷白；可加問號製造懸念。
- 生成提示詞（英文）：
```
cinematic documentary still, dark moody lighting, cold desaturated palette, film grain,
a dim evidence storage room, a dusty cardboard box holding an old murder case file folder
with a worn label reading the year "1986", faint police tape, in the background an out-of-focus
1980s suburban apartment exterior at night in silhouette (no modern landmarks, period-correct),
shallow depth of field, high contrast, single strong focal point, cold-case archival tone,
16:9 --ar 16:9
```

## 共同規範
- 不使用可辨識真實人臉；暗背景、單一強焦點、頻道一致字體色塊。
- 警徽不得出現可辨識真實單位字樣/徽記（用 generic），避免商標與法律風險。
- 鑑識畫面去識別、不血腥；暴力以暗示處理（對齊 legal-redlines 與 visual-style 禁忌）。
- 標題字 4–8 字、警示色，沿用頻道一致字體與色塊以建立辨識度。
