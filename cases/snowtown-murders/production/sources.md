# 來源清單 — 雪鎮殺人案 Snowtown Murders（影片描述引用用）

> 本表彙整 dossier.md 與 factcheck.md 之全部來源，供影片描述欄與片尾來源字卡引用。
> 「授權狀態」為素材使用面之初步判定，非法律意見；任何影像/影片/音檔素材於正式發布前須逐項向權利人確認。
>
> **授權狀態定義**
> - **公共領域 / 政府授權**：法院判決、官方政府文件等，內容事實可自由引用；澳洲政府著作多採 CC BY 4.0，仍應標註出處。
> - **CC BY-SA**：維基百科文字採 CC BY-SA 授權，引用須標示來源與相同方式分享；維基內嵌之圖片授權各異，須逐張查核。
> - **需授權**：新聞、商業媒體、true-crime 網站、影片、照片等受著作權保護之素材；**文字事實可引用，但其原創影像/影片/排版/配樂不可直接取用**，須取得授權。
> - **合理使用（引述事實）**：僅作為事實查核之佐證來源，影片中僅引用其陳述之「事實」而非其表達形式；不得搬運其圖文素材。
> - **低可靠性**：部落格 / 粉絲維基 / 播客摘要，僅供交叉參考，**不得作為唯一來源**，正式描述不建議單獨引用。

## 製作端注意事項（授權風險）

1. **影像 / 影片素材**：本案幾乎所有現有影像（嫌疑人照片、雪鎮銀行金庫、警方搜證畫面、《Snowtown》電影劇照）均受著作權保護，**屬「需授權」**。描述欄列出來源不等於取得使用權；片中若使用須逐項向權利人（通訊社、新聞台、電影發行商、NFSA）申請授權，或改用自製插畫 / 已授權圖庫。
2. **判決書與政府文件（A1–A4）**：內容事實可自由引用，澳洲政府著作多採 CC BY 4.0，引用時標註出處即可；惟正式全文須經 AustLII / 官方管道核校（dossier 標為待考項）。
3. **維基百科文字（B1–B4）**：採 CC BY-SA，引用文字須標示來源並以相同方式分享；**維基內嵌圖片授權各異，不可假設可用，須逐張查核**。
4. **The Conversation（E2）**：採 CC BY-ND，可整篇轉載但不可改作，須標示作者與出處。
5. **低可靠性來源（F 區）**：依 factcheck 結論，不得作為唯一事實依據，正式描述不建議單獨引用；僅在已有硬來源佐證時作交叉參考。
6. **高風險敘述**：依 factcheck「高風險主張」，描述欄與字卡涉及 Suzanne Allen 之死、Vlassakis 假釋狀態、被害確切日期、「227 名證人」等項時，須採法律語態與但書，避免斷言。
