# 案件真實畫面清單 — golden-state-killer（黃金州殺手案）

> **製作自律準則，非法律意見。** 本檔彙整黃金州殺手案（Golden State Killer / East Area Rapist / Original Night Stalker）真實素材的踏查結果，作為製作流程的素材風險控管參考。
> **本清單不構成法律意見，亦不代表任何素材可直接使用。** 任何素材在實際使用前，**一律須經人工審核並逐筆確認授權狀態 / 取得書面授權**，發布前一律人工確認授權。
> 對齊 `assets/copyright-guide.md`（三層用料優先序）、`assets/footage-sourcing.md`（第 3 層真實畫面）與 `brand/legal-redlines.md` §5 版權、§4 去識別（最高優先）。
> 本案涉性暴力與多名受害者，**配圖一律不還原真實受害者臉孔**（剪影／背影／馬賽克／示意，legal-redlines §4）。被告 Joseph James DeAngelo 已於 2020 認罪定讞，其官方 mugshot 屬公開法庭/警方紀錄、可作事實呈現；**受害者與倖存者一律去識別**。

## 風險分層總覽

依 copyright-guide 三層用料優先序，由風險低到高排列。**永遠先用第一層，用盡才考慮下一層。**

- **第一層 — 公共領域 / 官方機構**（聯邦作品、法院記錄、國家檔案館；風險：極低～低，優先採用）— 本案主力
- **第二層 — 可授權素材**（Commons CC-BY/CC-BY-SA、街景、歷史檔案館；風險：中，需逐檔查驗並取得授權）
- **第三層 — 受版權新聞 / 影視素材**（登錄供人工合理使用判斷，禁止未授權使用；風險：高，最後手段且最謹慎）

> **一個重要分界**：**聯邦**作品（FBI、NIST）依 17 U.S.C. §105 ＝公共領域；**加州州/地方政府**作品（如 Sacramento County Sheriff 的 mugshot、加州法院影像）**並不自動進入公共領域**，記錄雖公開但著作權可能仍存續，須**逐筆確認**。
