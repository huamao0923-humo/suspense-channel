# 版權與合理使用指南 — copyright-guide

> **製作自律準則，非法律意見。** 本檔為頻道內部素材使用的風險控管指引，協助製作流程在發布前自我把關。它**不能取代律師意見**；針對華語市場（尤其台灣較嚴格的法律環境），正式發布前的前 5 支影片建議經在地律師覆核。
> 本檔與 `brand/legal-redlines.md`（最高優先）對齊；兩者衝突時，以 legal-redlines 為準。

---

## 0. 核心原則（一句話）

**能不用受版權新聞素材就不用；非用不可時，只取必要片段、加實質分析、明確標出處、預期會被版權聲明並備好應對。**

合理使用是「**抗辯事由**」而非「**事前許可**」——它只有在被告上法院或在 YouTube 爭議程序中才被檢驗，因此每次使用都應假設未來需要為它辯護。

## 1. 三層用料優先序（A）

依風險由低到高排序。**永遠先用第一層，用盡才考慮下一層。**

### 第一層 — 公共領域 / CC0（風險：極低，優先採用）
完全迴避版權爭議，應作為素材主力。
- **政府公開素材／法庭影像**：警方公開影像、法庭錄影、起訴書/判決書截圖。
- **FOIA（資訊自由法）申請素材**：向美國聯邦/地方執法機構（FBI、警局）申請取得的公開檔案，通常無版權或屬公共領域。
- **公共領域照片/檔案**：Wikimedia Commons、Library of Congress、各國公開檔案館。
- **CC0 素材**：標示「No Rights Reserved」者，可自由使用。

### 第二層 — 授權素材（風險：低，需確認授權範圍）
取得明確授權或付費授權庫。
- **CC 授權新聞素材**：CC Search Portal、Al Jazeera Creative Commons 存檔等；**務必確認授權條款**（BY=須署名、NC=禁商用、SA=相同方式分享、ND=禁改作）。本頻道為商用（YouTube Partner），**不可使用 NC 條款素材**，ND 條款則不可改作。
- **付費授權庫**：Storyblocks、Artlist 等，提供紀錄片/教育用途授權的新聞風格素材。
- **向新聞機構/警局直接申請授權**：許多機構願授予教育/評論用途；保留授權書面紀錄。

### 第三層 — 受版權新聞素材（風險：中高，最後手段且最謹慎）
電視台新聞片段、新聞機構照片等，**僅在前兩層無法滿足敘事需求時使用**，且必須完整套用本檔第 3 節降風險清單，並預期觸發 Content ID（見第 4 節）。
- 他人製作的**紀錄片/影片一律不得使用**（對齊 legal-redlines §5）。
- 每項此類素材必須在製作包 `sources.md` 記錄來源與授權狀態，並標記「依合理使用主張使用」。

## 2. 合理使用四要素與引用界線（B）

合理使用源自美國版權法 17 USC §107，法院綜合衡量四要素。**沒有任一要素是決定性的**，但「目的及性質（轉化性）」實務上權重最高。

| # | 要素 | 對本頻道有利的做法 | 不利（高風險）的做法 |
|---|------|------------------|--------------------|
| 1 | **目的及性質（轉化性 Transformative）** | 加入實質評論、調查分析、新觀點、新脈絡；以「教育/紀實/分析」框架呈現 | 單純冗長重放新聞片段、邊看邊講閒話、無新增意涵 |
| 2 | **著作性質（Nature）** | 使用事實性新聞報導（事實本身不受著作權保護） | 取用含創意元素者（新聞畫面的攝影視角、配樂、編輯構思） |
| 3 | **質量及比例（Amount）** | 取用足以支撐評論的**最少必要範圍**；3–5 秒短片分散使用 | 取用原作「心臟部分」（the heart of the work）；單一長片段 |
| 4 | **市場效應（Market Effect）** | 評論原案件、創造全新需求，不取代原新聞市場 | 取代原機構素材市場、用於原機構靠販售獲利的素材 |

**引用界線速記：**
- ✅ 必要範圍內（within necessary scope）：只取支撐當下分析所需的最少素材。
- ✅ 多個 3–5 秒短片 + 緊接實質評論，分散於全片。
- ❌ 連續播放 5 分鐘以上新聞直播。
- ❌ 純新聞重放而無評論。
- ❌ 以「只是分享」「非營利」為由（頻道已開啟 Partner Program 即屬商用）。

**事實性新聞免責特例（台灣）**：台灣著作權法第 9 條第 1 項第 4 款規定「單純為傳達事實之新聞報導所作成之語文著作」不為著作權標的——即純事實陳述可自由引用，但**含編輯/評論/攝影創意的新聞製作仍受保護**。

## 3. 降風險清單（C）

每次使用第三層（受版權）素材，**逐項確認**：

- [ ] **僅用必要片段**：新聞素材切成 3–5 秒短片，分散全片，不長播。
- [ ] **加實質分析評論**：每個片段後立即添加解讀、質疑、背景脈絡或新調查發現。用「此新聞遺漏的關鍵問題是…」「後續調查發現…」，而非閒聊式旁白。
- [ ] **轉化**：確保整體呈現添加了新的表達、意涵或評論，而非單純複製。
- [ ] **重繪/動畫替代**：優先以自製動畫、手繪重現場景、動畫地圖/時間軸取代直接播放新聞畫面（高轉化性，且對齊 visual-style）。
- [ ] **模糊化**：基於隱私/倫理需要時打馬賽克或模糊——但**注意：模糊不等於合理使用**（曾有像素化後仍被判侵權、和解賠款 $32,500 的案例）。模糊只服務去識別，不可當版權防線。
- [ ] **標出處**：精確標示拍攝者/機構名稱與日期（例「新聞來源：CNN 2023-03-15」）。**標出處有益但非護身符**，無法把違法使用變合法；台灣著作權法第 64 條要求明確標示著作人、名稱、出處，「翻拍自 YouTube」不足。
- [ ] **記錄於 sources.md**：來源、日期、授權狀態、合理使用主張理由。

> **關鍵提醒**：單純「像素化/模糊」或加「版權屬於原作者」聲明，**均不足以**構成轉化性或豁免責任。降風險靠的是「必要片段 + 實質評論 + 轉化」的組合，而非單一動作。

## 4. YouTube Content ID / 版權聲明的風險與處理流程（D）

### 風險認知
- **Content ID 是機器，不懂脈絡**：自動掃描比對，只判斷時長與匹配比例，**無法識別教育用途、評論或轉化性**。即使合理使用成立，影片仍可能被立即阻擋、去推薦、失去初期流量。
- **新聞素材特別容易觸發**：多數新聞機構已註冊內容到 Content ID 資料庫，使用第三層素材應**預期會收到版權聲明**。
- 匹配後版權方可：(1) 全面阻擋觀看；(2) 加廣告並收走收益；(3) 僅追蹤統計。

### 處理流程（收到 Content ID 聲明時）

1. **判斷是否合理使用** → 檢查：是否僅必要片段？是否有實質評論？四要素是否站得住？
2. **若主張合理使用 → 30 天內提起爭議（dispute）**，附上合理使用聲明文件（引用本檔四要素分析 + sources.md 紀錄）。爭議期間影片通常恢復上線。
3. **版權方有 7 天回應**：
   - 釋出聲明 → 結案。
   - 維持/拒絕 → 可進一步**上訴（appeal）**，但程序冗長；即使最終勝訴，初期流量與演算法懲罰已造成。
4. **若主張站不住 → 不要硬爭議**：改採行動以止損——換成第一/二層素材重新上傳、剪掉爭議片段，或接受收益分配。
5. **帳戶保護**：避免累積多次失敗爭議或版權警告（strike），多次可能招致額外帳戶限制。

### 應急預案（事前準備）
- 所有素材出處即時記錄於 `sources.md`。
- 每支使用第三層素材的影片，預先撰寫一份合理使用聲明草稿。
- 收到任何**正式法律信函或投訴** → 依 legal-redlines §7，**立即下架**相關內容並保留所有來源紀錄（此優先於版權爭議流程）。

## 5. 風險矩陣表（E）

| 素材類型 | 風險等級 | 主要對策 |
|---------|---------|---------|
| 完全自製動畫/重繪/手繪 | 極低 | 直接使用；高轉化性，亦對齊 visual-style |
| FOIA 取得的執法機構素材 | 極低 | 直接使用；記錄 FOIA 來源 |
| 公共領域 / CC0 照片檔案 | 極低 | 直接使用；確認確為 PD/CC0 |
| 政府公開影像 / 法庭影像 | 低 | 標出處；確認非受限發布 |
| CC 授權新聞素材（BY/BY-SA） | 低 | 依條款署名；**避開 NC/ND**（本頻道商用且需可改作） |
| 付費授權庫素材 | 低 | 確認授權涵蓋商用與 YouTube 散布 |
| 事實性純新聞文字 | 低 | 可引用事實；改寫為自己的敘述並標來源 |
| 受版權新聞片段（3–5 秒 + 實質評論） | 中 | 必要片段 + 轉化評論 + 標出處 + 預期 Content ID 爭議準備 |
| 受版權新聞照片（單張，加分析） | 中 | 必要範圍 + 評論 + 標出處；優先考慮重繪替代 |
| 受版權新聞長片段（無評論/長播） | 高 | **避免**；改重繪/動畫或取得授權 |
| 他人製作的紀錄片/影片 | 高 | **禁用**（除非取得明確授權，對齊 legal-redlines §5） |
| 模糊/像素化的受版權素材（當唯一防線） | 中高 | 模糊不豁免版權；須另備完整合理使用主張，否則改第一/二層 |

> **地域提醒**：同一影片在美國可能成立合理使用，在**台灣**（著作權法第 65 條較僵化、商用獲利常逆轉有利因素）可能被判侵權，**香港**（fair dealing，定性判準、有新聞豁免）介於兩者之間。**面向華語市場應採最嚴格標準（台灣）自我檢視。**

## 6. 與 legal-redlines 對齊：去識別與不還原真實人臉（F）

版權與隱私是兩條獨立紅線，**同時成立才可使用**：

- **去識別優先於版權考量**：未成年人（加害/受害皆然）完全去識別用代號；受害者/倖存者避免公開臉部、全名、住址、病歷、性暴力細節（legal-redlines §4）。
- **不還原真實人臉**：配圖/素材**一律不呈現可辨識的真實受害者/嫌疑人臉孔**，用剪影、背影、馬賽克或示意動畫（legal-redlines §4、visual-style 禁忌）。
- **重繪/動畫的雙重好處**：自製重繪既提高版權上的轉化性（降版權風險），又能做到去識別（符合隱私紅線）——是處理敏感新聞畫面的**首選替代方案**。
- **模糊的定位**：模糊真實人臉主要服務「去識別」隱私目的；它**不**自動構成版權合理使用，兩條紅線需各自滿足。
- **推定無罪同步把關**：素材若顯示未定罪者，旁白與字卡須用「涉嫌/被指控」，不得因畫面而暗示有罪（legal-redlines §1）。

## 製作前自檢清單（使用任何外部素材前逐項過）

- [ ] 是否已優先嘗試第一層（PD/CC0）、第二層（授權）素材？
- [ ] 用第三層（受版權）素材的理由是否「敘事必要」且已記錄？
- [ ] 是否僅取必要片段（3–5 秒短片、分散）？
- [ ] 每個片段是否緊接實質分析/評論（非閒聊）？
- [ ] 是否可改用重繪/動畫替代？
- [ ] 出處是否精確標示（機構名 + 日期）並寫入 sources.md？
- [ ] 是否預期 Content ID 聲明並備妥合理使用聲明草稿？
- [ ] 是否同時滿足去識別與不還原真實人臉（對齊 legal-redlines §4）？
- [ ] CC/授權素材是否避開 NC/ND、符合商用？
- [ ] 是否以「教育/紀實/分析」框架呈現，不美化犯罪？
