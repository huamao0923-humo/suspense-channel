# 案件真實畫面清單 — golden-state-killer（黃金州殺手案）

> **製作自律準則，非法律意見。** 本檔彙整黃金州殺手案（Golden State Killer / East Area Rapist / Original Night Stalker）相關真實素材的搜尋結果，作為製作流程的素材風險控管參考。
> **本清單不構成法律意見，亦不代表任何素材可直接使用。** 任何素材在實際使用前，**一律須經人工審核並逐筆確認授權狀態 / 取得書面授權**。
> 對齊 `assets/copyright-guide.md`（三層用料優先序）、`assets/footage-sourcing.md`（第 3 層真實畫面）與 `brand/legal-redlines.md` §5 版權、§4 去識別（最高優先）。
> 本案涉性暴力與多名受害者，**配圖一律不還原真實受害者臉孔**（剪影／背影／馬賽克／示意，legal-redlines §4）。被告 Joseph James DeAngelo 已於 2020 認罪定讞，其官方 mugshot 屬公開法庭/警方紀錄、可作事實呈現；**受害者與倖存者一律去識別**。

---

## ◆ 本案素材特性（與雪鎮的關鍵差異）

黃金州殺手案的真實素材**第一層特別豐沃**：本案由 **FBI（聯邦調查局）** 長期主辦並大量公開素描、地圖、證物照、記者會與通緝影片。依美國著作權法 **17 U.S.C. §105，聯邦政府職務作品不受著作權保護＝公共領域**。因此本案**大半畫面可直接用第一層 PD 素材撐起，幾乎不必動用高風險的第三層受版權新聞片段**——這是 GSK 作為「真實素材導向」選題的最大優勢。

> **一個重要分界**：**聯邦**作品（FBI、NIST）＝公共領域；**加州州/地方政府**作品（如 Sacramento County Sheriff 的 mugshot、加州法院影像）**並不自動進入公共領域**（加州無等同 §105 之規定，記錄雖公開但著作權可能仍存續）。故 Sheriff mugshot、加州法庭畫面須**逐筆確認**，不能一律當 PD。

## 風險分層總覽

依 copyright-guide 三層用料優先序，由風險低到高排列。**永遠先用第一層，用盡才考慮下一層。**

- **第一層 — 公共領域 / 官方公開素材**（風險：極低～低，優先採用）— 本案主力
- **第二層 — 可授權素材**（風險：低～中，需確認授權範圍並取得授權）
- **第三層 — 受版權新聞 / 影視素材**（風險：中～高，最後手段且最謹慎）

### 受版權素材「合規使用建議」（逐項對齊 copyright-guide 第 3 節）

1. **僅取必要片段**：新聞/庭審片段切 3–5 秒、分散，不長播。
2. **加實質分析**：每段後立即接解讀、質疑、後續調查（拉高轉化性，fair use 權重最高）。
3. **明確標出處**：機構名＋日期（例「ABC10 2020-08-21」），寫入製作包 `sources.md`。
4. **優先公共領域替代**：本案能用第一層（FBI PD 素描/地圖/證物/記者會、CC0 對比圖）就**不用本層**——這是 GSK 的核心優勢。
5. **必要時改用重繪/動畫替代**：自製動畫地圖、家系樹流程、時間軸取代直接播放新聞畫面（高轉化性＋去識別）。模糊只服務去識別，不等於版權豁免（兩條紅線各自滿足）。
6. **預期 Content ID 並備妥合理使用聲明草稿**。
7. **HBO 紀錄片特別注意**：影視製作品，**禁用**（除研究參考）；首選重繪或第一層替代。
8. **地域最嚴標準**：面向華語市場以台灣最嚴格標準自我檢視。

### C. 建議重繪 / 動畫替代（高轉化性 + 去識別，首選方案）
