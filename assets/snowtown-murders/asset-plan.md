## ◆ 全片合規紅線（每段適用，先讀）

源自 legal-redlines.md 與 broll-list 總則，**任一違反一律重做**：

1. **無可辨識真實人物臉孔**：被害人、被告、證人一律剪影 / 背影 / 像素化 / 裁切；不還原真實長相。
2. **無血腥、無遺骸、無解剖寫實**：以環境、物件、檔案、留白暗示；醫學圖一律「示意圖 schematic」。
3. **無寫實食人 / 分屍描寫**（03d David Johnson 案最嚴）：以空間與文件暗示嚴重性。
4. **未成年 / 受害者去識別**：依最嚴標準，無可辨識特徵。
5. **無可辨識真實地址 / 門牌 / 車牌**：真實街景去識別化，或改用通用郊區建築。
6. **未定讞主張用法律語態**：本案主犯已定讞可援引判決書；Suzanne Allen 之死、Vlassakis 假釋、確切被害日期、「227 名證人」等須加但書（見 factcheck 高風險清單）。
7. **片尾統一聲明**：AI 生成畫面 + 真實素材來源與授權方式 + 示意圖標註。
