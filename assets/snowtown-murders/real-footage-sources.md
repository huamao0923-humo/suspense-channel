# 案件真實畫面清單 — snowtown-murders（雪鎮謀殺案）

> **製作自律準則，非法律意見。** 本檔彙整雪鎮謀殺案（Snowtown Murders）相關真實素材的搜尋結果，作為製作流程的素材風險控管參考。
> **本清單不構成法律意見，亦不代表任何素材可直接使用。** 任何素材在實際使用前，**一律須經人工審核並逐筆確認授權狀態 / 取得書面授權**。
> 對齊 `assets/copyright-guide.md`（三層用料優先序）、`assets/footage-sourcing.md`（第 3 層真實畫面）與 `brand/legal-redlines.md` §5 版權、§4 去識別（最高優先）。
> 本案涉嚴重暴力與受害者，**配圖一律不還原真實受害者/嫌疑人臉孔**，未定罪者稱呼用「涉嫌/被指控」（legal-redlines §1、§4）。

---

## 風險分層總覽

依 copyright-guide 三層用料優先序，由風險低到高排列。**永遠先用第一層，用盡才考慮下一層。**

- **第一層 — 公共領域 / 官方公開素材**（風險：極低～低，優先採用）
- **第二層 — 可授權素材**（風險：低～中，需確認授權範圍並取得授權）
- **第三層 — 受版權新聞 / 影視素材**（風險：中～高，最後手段且最謹慎）

### 受版權素材「合規使用建議」（逐項對齊 copyright-guide 第 3 節）

對上表每一項受版權素材，使用前**逐項確認**以下降風險組合（單一動作不足以豁免，須組合套用）：

1. **僅取必要片段**：新聞片段切成 3–5 秒短片、分散全片，不長播；新聞照片只取支撐當下分析所需的最小範圍。
2. **加實質分析**：每個片段/照片後立即添加解讀、質疑、背景脈絡或後續調查發現（如「此報導遺漏的關鍵問題是…」），而非閒聊式旁白——拉高轉化性（fair use 權重最高的要素）。
3. **明確標出處**：精確標示機構名與日期（例「新聞來源：ABC News 2020」「7News」「9News」），並寫入製作包 `sources.md`。標出處有益但非護身符，不能把違法使用變合法（台灣著作權法第 64 條要求明確標示著作人/名稱/出處）。
4. **優先公共領域 / CC 替代**：能用第一層（Google 街景、SLSA 1955 前照片、Wikipedia 文字）或第二層（**ABC CC BY-SA 開放檔案**）就不用本層；特別是 ABC 新聞片段，應優先尋找其 CC 釋出版本。
5. **必要時改用重繪 / 模糊 / 動畫替代**：優先以自製動畫、手繪重現、動畫地圖/時間軸取代直接播放新聞畫面或電影片段（高轉化性，且對齊 visual-style 與 legal-redlines §4 去識別）。
   - **模糊只服務去識別，不等於版權合理使用**（曾有像素化後仍被判侵權、和解 $32,500 的案例）；版權與隱私是兩條獨立紅線，須各自滿足。
   - **不還原真實受害者/嫌疑人臉孔**：用剪影、背影、馬賽克或示意動畫（legal-redlines §4）。
6. **預期 Content ID 並備妥應對**：新聞機構多已註冊 Content ID，使用本層素材應預期收到版權聲明；預先撰寫合理使用聲明草稿（引用四要素分析 + sources.md 紀錄）。
7. **電影《Snowtown》特別注意**：屬商業劇情片（非新聞、非事實素材），合理使用空間最小；除極短評論性摘錄（約 5 秒）外，**首選重繪替代或正式授權**。
8. **地域最嚴標準**：面向華語市場以台灣（著作權法第 65 條較僵化、商用獲利常逆轉有利因素）的最嚴格標準自我檢視。

## 待辦

### A. 可直接用（風險極低～低，仍須最終人工確認）
- **Google 街景**（Snowtown Fourth St / State Bank 建築）：依 Google Maps API 政策嵌入。
- **ABC CC BY-SA 開放檔案**：CC BY-SA 下可商用，僅需正確署名（「ABC / Creative Commons BY-SA」）並對衍生採相同授權——**本案新聞素材的首選**，發布前確認所取素材確屬此 CC 釋出範圍。
- **Wikipedia 條目文字**（CC-BY-SA）：可改寫引用並署名（圖像須另查）。
- **SLSA 1955 年前歷史照片**：確認確為 1955 前公共領域者可用。

### B. 需申請授權 / 逐筆確認（不可未經授權即用）
- **NFSA 素材庫**（含「Bodies in the Barrels」）：向 NFSA 申請；澳洲獨立製作者可試「Take Three / Zero Fee」免費管道。
- **SA Police FOI**：以 PD360 表格申請 1998–1999 調查紀錄/警方影像（注意部分可能因隱私遭拒）。
- **Trove / AIDR 知識庫**：逐項追查原始來源授權。
- **Flickr / Tripadvisor / Silo Art Trail 地點照片**：逐筆確認發佈者授權（避開 NC/ND）。
- **ABC / 7News / 9News 新聞片段**：商業發行須向各台申請；若主張合理使用須完整套用降風險清單。
- **電影《Snowtown》**：任何摘錄須申請授權或嚴格限極短評論性引用。

### C. 建議重繪 / 動畫替代（高轉化性 + 去識別，首選方案）
- **電影《Snowtown》畫面**：以自製重現/動畫取代直接摘錄（商業劇情片合理使用空間最小）。
- **任何含真實受害者/嫌疑人臉孔的新聞照片/片段**：一律改剪影、背影、馬賽克或示意動畫（legal-redlines §4）。
- **無 CC/公共領域版本的新聞畫面**：優先以動畫地圖、時間軸、手繪重現案發地點與經過取代直接播放。
- **YouTube 第三方真實犯罪頻道內容**：不直接引用；參考其角度後自行蒐集原始素材重新製作。
