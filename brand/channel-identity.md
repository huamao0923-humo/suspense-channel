# 頻道定位 — channel-identity

> 草稿。頻道名稱與 tagline 待你拍板（標記 ⬜ 待定）。

## 一句話定位
繁體中文、faceless 的**國外懸案推理長片**頻道。用嚴謹考證 + 理性推理 + 尊重受害者的敘事，帶觀眾一步步看懂一樁懸案。

## 頻道名稱與標語
- 頻道名稱：**Pilot 調查員**
- Tagline：**撥開迷霧，領航真相。**
- 命名意象：Pilot＝領航員。每一集帶觀眾「飛入」一樁懸案的迷霧核心，冷靜領航、層層推理，最後落在真相（與真相的界線）上。

## 目標受眾
- 主要：台灣、香港、新加坡的繁中觀眾
- 輪廓：喜歡 true crime / 推理 / 紀錄片、習慣看 20–50 分鐘長片、重視考證與邏輯勝過獵奇

## 內容支柱（題材類型）
1. 國外知名未解失蹤案
2. 國外離奇/高戲劇張力命案
3. 國外連環/連續犯罪
4. 近期翻案、有新事證或新判決的舊案

## 選題標準（WF1 評分依據）
| 維度 | 加分 | 扣分 |
|---|---|---|
| 戲劇張力 | 反轉、謎團、未解 | 平淡、結論顯而易見 |
| 資料充足度 | 公開判決/新聞/維基齊全 | 來源稀少、難查證 |
| 獨特性 | 中文圈少被深做 | 已被對標頻道做爛 |
| 法律隱私風險 | 國外、已定讞、年代較遠 | 涉未成年、進行中、未定罪爭議 |
| 中文受眾共鳴 | 主題普世、有人性張力 | 過度在地化、文化隔閡大 |

## 差異化（怎麼跟對標頻道不一樣）
- 更明確的「事實 vs 推測」標示，建立可信度
- 推理過程透明：把線索、矛盾點攤開讓觀眾一起推
- 尊重受害者的敘事基調，不靠血腥縮圖衝點擊

## 影片規格
- 長度：30–50 分鐘（長片為主）
- 更新：起步建議雙週更，穩定後週更
- 形式：faceless 旁白 + 模擬重演/配圖 + 地圖時間軸
